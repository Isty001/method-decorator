# MethodDecorator

Method decorator implementation for Ruby, similar to Python. 

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'method-decorator'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install method-decorator
